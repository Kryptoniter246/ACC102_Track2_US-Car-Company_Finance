## ACC102 Track 2 – US Automakers Financial Data Analysis Project

## 1. Product & User
This data product helps **car buyers and auto enthusiasts** understand the financial performance and stability of major U.S. automakers. By analysing **historical data and financial trends**, users can evaluate brand reliability, long‑term stability, and financial strength before making purchasing decisions.

## 2. Data
- Source: WRDS Compustat
- Period: 2020 – 2024 (5 full fiscal years)
- Companies: Ford (F), General Motors (GM), Tesla (TSLA), Rivian (RIVN), Lucid (LCID)
- Variables: Net Income, Revenue, Operating Profit, Net Profit Margin

## 3. Methods
- Connect to WRDS and extract 5-year financial data using SQL.
- Clean, rename columns, and calculate financial indicators.
- Use loops and input function  to support interactive selection of any companies.
- Generate profit tables and financial analysis (average, max/min, CAGR).
- Plot profit trend charts and export images.
- Export raw data and analysis results to Excel following class PPT method.

## 4. Key Findings 
- Traditional automakers (Ford, GM) show stable financial performance.
- Tesla maintains strong and consistent net income growth.
- EV startups (Rivian, Lucid) face higher profit volatility and challenges.
- Profit trends reflect long-term brand reliability and operational strength.

## 5.How to run
Run the main Python notebook:  
**`notebook.ipynb`**

STEPS:
1. Open `notebook.ipynb` in Jupyter Notebook / JupyterLab.
2. Replace `your_wrds_username` with your personal WRDS username.
3. Run all cells sequentially.
4. When prompted, enter the tickers you want to analyse (e.g., `F,TSLA` or `GM,RIVN,LCID`).
5. The system will display:
   - 5-year net income table
   - Financial performance analysis (average, max/min, CAGR)
   - Profit trend chart
6. The product automatically exports:
   - `profit_trend.png`
   - `automakers_raw_data.xlsx`
   - `automakers_analysis_summary.xlsx`

### Important Note
